2019/04/14 HTTPS対応ついでにcloudfrontを導入してみた

2019/04/14 17:01:10
tech.てんてーちぶ.
 

遅れること早数年、当ブログもHTTPSに対応した。

ローカルで動かしてるブログを静的HTMLにコンバートして、S3にアップロードする形で動かしているんだけれど、今回はS3の前段にcloudfrontを置いた形。


◆ Cloudfrontの設定でハマったところ

Origin Domain NameのところにS3でホスティングしている場合、
Static WebSite Hostingで確認出来るエンドポイントを指定しないと上手く動かなかった。

これに気づくまで時間がかかったけれど、それ以外の設定は本当に簡単でとても気軽にCDNを入れれてHTTPSでも見れるようになったしAWSサイコーだな〜という気持ちになった。

Page Speed Insightsで計測した感じ、スコアも上がっていたのでパフォーマンスの改善も期待出来そうな雰囲気。お仕事にも導入してみようかな。
