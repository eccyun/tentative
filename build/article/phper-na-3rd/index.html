ぺちぱな。つぎのつぎ〜 use git as scm; 〜
http://atnd.org/events/39393

掲題のイベントにスタッフとして参加させていただきました。

自分の役割は実習時に参加者が困ったりわからないことがあったときのサポートとgitの活用例について10分くらいの時間でお話をすることでした。

今回は決定から開催まで比較的時間がありましたが、当日の3日前くらいまでお話しする内容もまとまらず個人的にもブルーになってたところもあったので、大丈夫なのかなぁとかとても心配でした。

アドイバスを頂きながら微調整を入れていき、なんとかスライドが仕上がったのが前日でそれからは声に出して部屋で練習したりしていました。

練習した成果もあってか (?) 評判は今までで一番良かったです。
形式がちょっと特徴的だったところもあるかもしれませんが、当日までじっくりgitを触ることが出来たのも大きかったかもしれません。
個人的には今回でようやくなんとか...といった感じでした。

人に教えるってやっぱり楽しいなぁと思う。
自分がやったことをこれからも誰かの為に伝えていけるといいなぁ。
